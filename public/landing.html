快速配置指南

  1. 获取 DeepSeek API Key：访问 platform.deepseek.com/api_keys 注册并创建 API Key（费用极低，约 ¥1/百万 token）
  2. 打开应用：安装后启动"牙科知识 AI 问答"
  3. 配置 API Key：点击右上角设置 ⚙️，粘贴你的 DeepSeek API Key，点击"测试连接"
  4. 开始学习：选择一个题库，输入你的回答，AI 会自动评判
  5. 联网搜索增强（可选）：在设置中开启"联网搜索增强"，AI 判分前会通过百度搜索获取最新医学参考资料

💡 提示：即使没有 API Key，应用也能使用本地关键词判分引擎正常学习。所有数据保存在你的设备上。
