========================================
  牙科知识 AI 问答 v1.0.0
  使用说明
========================================

📦 文件说明
-----------
  牙科知识AI问答.html  →  网页版（双击用浏览器打开，无需安装）
  牙科知识AI问答.exe   →  Windows 桌面版（双击安装/运行）

🌐 网页版使用方法（推荐，最简单）
-----------------------------------
  1. 双击 "牙科知识AI问答.html"
  2. 浏览器中打开后即可开始学习
  3. 所有数据保存在浏览器本地，不会上传

🪟 Windows 桌面版使用方法
-----------------------------------
  1. 双击 "牙科知识AI问答.exe" 启动应用
  2. 首次启动可能需要几秒钟

⚙️ 配置 AI 判分（可选，但推荐）
-----------------------------------
  AI 判分可以让系统语义理解你的回答，而不要求一字不差。

  1. 在应用右上角点击 ⚙️ 设置
  2. 选择 "DeepSeek V3" 作为判分引擎
  3. 输入你的 API Key（在 platform.deepseek.com 免费注册获取）
  4. 点击 "测试连接" 确认可用
  5. 可以开启 "联网搜索增强" 获得更精准的判分参考

  💰 费用说明：DeepSeek API 极低价，约 ¥1/百万 token
     每次判分约消耗 500-2000 token，成本约 ¥0.001-0.002

📡 离线使用
-----------
  即使没有 API Key，应用也能使用本地关键词判分引擎工作。
  所有学习记录、题库数据都保存在你的设备上。

📚 功能简介
-----------
  - 牙科知识题库（内置 20 张卡片，覆盖 10 个领域）
  - AI 语义判分 / 本地关键词判分
  - DOCX 文档导入（支持多种格式解析）
  - 错题本自动收集
  - 题库管理（创建/编辑/删除）
  - 学习记录与统计

-----------

📝 系统要求
-----------
  网页版：任意现代浏览器（Chrome / Edge / Firefox / Safari）
  Windows 版：Windows 10/11，需要 WebView2 运行时
              （Windows 11 已内置，Win10 会自动安装）
